payment-gateway plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/NekoNeko6996/tutor-contrib-payment-gateway.git

Usage
*****

.. code-block:: bash

    tutor plugins enable payment-gateway
